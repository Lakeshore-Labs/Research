# Research Proposals

A set of self-contained research projects across AI/ML/SW. Each proposal is a standalone markdown doc with a fixed structure: motivation, research questions, scope, suggested approach, deliverables, a **runnable demo**, success criteria, prerequisites, lightweight phases, stretch goals, and starter references.

All projects are scoped to be **demoable**: every one ends in something runnable (a script, notebook, endpoint, or dashboard), regardless of whether the primary output is a prototype, benchmark, report, or internal tool.

## How to use these

Pick one per person (or pair, for the Advanced ones). Each doc is the starting brief; the **Open Questions / Assumptions** section at the bottom of every proposal lists decisions to settle before kickoff (budgets, datasets, target thresholds).
